# 🚀 ISpooferMotion v2.4.0 (2026-07-12)

## ✨ Highlights

- **UI Modernization:** Interface utilizes standard `shadcn` components. The new design is much cleaner (Layout is the same, but with shadcn components).
- **Manual Swap Mode:** The Roblox Studio plugin now features a dedicated **Manual Swap Mode**. You can manually map original asset IDs to spoofed IDs if you prefer not to use automatic patching.
- **Video Download Only:** Video assets are now strictly downloaded, preventing accidental video uploading and removing configuration clutter.

## 🛠 Improvements & Polish

- **UI Polish:** Fixed UI clipping, aggressive layout borders, and hidden scrollbar issues in the Asset Explorer and Debug Console. The Asset Explorer no longer automatically selects all items on load.
- **Settings Persistence:** Fixed a bug where locally cached Cookies and API keys would fail to restore when the application restarted.

## 🐛 Bug Fixes

- **Studio Plugin Stability:** Resolved connection loss and failure to apply auto-replacements after extended processing jobs.
- **Instant Patching:** Fixed a bug where the desktop daemon silently queued replacements. The Studio plugin now instantly receives patches without artificial delays.
- **Long-Poll Timeouts:** Lowered the daemon's long-poll timeout from 25s to 8s to prevent Roblox Studio's `HttpService` from timing out during heavy workloads.
- **Execution Metrics:** Adjusted Pause/Resume logic to accurately halt execution ETA timers instead of infinitely accumulating.
- **Clipboard Permissions:** Fixed an issue preventing the clipboard manager from reading text by properly declaring read permissions in the Tauri capabilities configuration.

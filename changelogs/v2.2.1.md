## [2.2.1] - 2026-07-11

### ✨ What's New

- **Unified Retry System**: The "Retry Failed" button now automatically tracks and retries any assets that failed to apply directly within Roblox Studio, giving you a second chance at missed replacements without starting over.
- **Persistent Studio UI**: The plugin window in Roblox Studio will no longer close automatically after completing a scan or applying assets, giving you time to review the final status.
- **Smoother App Updates**: Improved the updater window experience. The styling now remains consistent while downloading an update, and visual glitches have been removed.

### 🐛 Bug Fixes & Under the Hood

- **Accurate Asset Replacements**: Fixed a critical bug where short asset IDs (like `12345`) could inadvertently corrupt longer IDs (like `123456789`) during replacement. Assets are now applied perfectly every time.
- **Faster Studio Connections**: The app now connects to Roblox Studio noticeably faster and recovers more reliably during rapid reconnects.
- **Safer Tag Handling**: Fixed an issue where unrelated object tags could be modified when spoofing assets, ensuring your existing game logic remains untouched.
- **Folder Opening**: Opening system folders from within the app is now completely native and significantly more reliable.
- **Responsive Background Tasks**: Improved responsiveness throughout the application, especially during heavy file operations and plugin synchronizations.
- **Overall Stability**: Eliminated edge cases that could cause the app to freeze when handling rapid network requests.
- **Reduced App Overhead**: Significantly improved background processing speed and optimized network usage, reducing the app's overall resource footprint.

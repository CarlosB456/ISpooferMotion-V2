## [2.3.0] - 2026-07-11

### ✨ What's New & Improved

- **Faster Uploads & Scraping**: Uploading assets, scraping creators, and looking up games should now be noticeably faster.
- **Lower Memory Usage**: Handles large asset files much more efficiently instead of loading everything into memory at once. This means less RAM usage and better performance during larger spoofing jobs.
- **Smoother Background Tasks**: File checks, cache reading, and other heavier tasks should no longer cause random UI freezes or stutters. More of this work now happens properly in the background.
- **Complete Documentation**: Codebase has been thoroughly documented internally, making it significantly easier to maintain, debug, and build upon, if you so choose.

### 🐛 Bug Fixes & Improvements

- **Better Rate Limit Handling**: Improved how Roblox rate limits and network timeouts are handled. Large upload jobs and lookups should be much less likely to stall or randomly fail.
- **Cleaner Diagnostics**: Failed upload diagnostics are now cleaned up more reliably, helping prevent unnecessary files from building up in the background.
- **Faster Caching**: Improved how places and assets are cached locally, reducing unnecessary disk work and speeding up repeated tasks.

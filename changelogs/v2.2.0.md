## [2.2.0] - 2026-07-09

### ✨ What's New

- **Native Place Parsing**: Moved `.rbxl` and `.rbxlx` parsing out of TS web workers and into the Rust backend. This fixes UI thread freezes on massive places and speeds up load times.
- **UI Layout Changes**: Removed the dedicated Experimental and Place ID Selector views. Settings are now split between a centralized Config view and an Advanced Settings modal.
- **Standalone Updater UI**: Created a separate lightweight window (`splash.html`, `splash.ts`) specifically for the auto-updater.
- **Component Refactor**: Broke down massive React files (`SpoofingView`, `App`) into smaller, focused components (`SpoofingControls`, `AnimationPreview`, `RobloxStatusBanner`, `ExecutionLogs`).
- **App Initialization**: Refactored app bootstrapping and state synchronization into a dedicated `useAppInitialization` hook.
- **Translation Updates**: Dropped dead localization keys from older UI versions across all supported languages (EN, ES, FR, RU) and synced missing translations.

### 🐛 Bug Fixes & Under the Hood

- **Studio Bridge**: Tweaked the messaging middleware and API routing to make Roblox Studio connections more stable.
- **Backend Architecture**: Refactored the monolithic `resolver` file by extracting Roblox Open Cloud API logic into a dedicated `roblox_api` domain module.
- **Type Safety**: Centralized auto-generated Rust-to-TypeScript IPC bindings into `src/types/bindings.ts` to keep the root directory clean.
- **Error Handling**: Eliminated empty error-swallowing `catch` blocks in TypeScript files, replacing them with explicit warnings to improve debuggability.
- **Configuration Resilience**: Fixed a bug where a mismatched default value (`activeTab`) could cause a blank screen, and added safe fallbacks for reading cached Studio states.
- **Stricter Linting**: Removed leftover `any` casts in the frontend and updated `selene.toml` to fix unscoped variable spam and concatenated file scoping in the Luau plugin.
- **CI/CD Reliability**: Replaced the deprecated `setup-selene` GitHub Action with native `cargo install` and migrated to using the GitHub CLI for explicit release note updates.
- **Dependencies**: Pinned NPM packages to exact versions to prevent random upstream breakages, and pruned unused Cargo dependencies.
- **Performance**: Reduced React re-renders by batching UI updates when streaming large DOM trees from the backend.

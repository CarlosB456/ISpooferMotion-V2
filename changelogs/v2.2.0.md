## [2.2.0] - 2026-07-09

### ✨ What's New

- **UI Layout Changes**: Removed the dedicated settings page. Settings are now split between a centralized Config view and an Advanced Settings modal.
- **Native Place Parsing**: Moved `.rbxl` and `.rbxlx` parsing out of TS web workers and into the Rust backend. This fixes UI thread freezes on massive places and speeds up load times.
- **Component Refactor**: Broke down massive React files into smaller, focused components (`SpoofingControls`, `AnimationPreview`, `RobloxStatusBanner`).
- **Translation Updates**: Dropped dead localization keys from older UI versions and synced missing translations across all languages.

### 🐛 Bug Fixes & Under the Hood

- **Studio Bridge**: Tweaked the messaging middleware and API routing to make Roblox Studio connections more stable.
- **Stricter Linting**: Removed leftover `any` casts in the frontend and updated `selene.toml` to fix unscoped variable spam in the plugin.
- **Dependencies**: Pinned NPM packages to exact versions to prevent random upstream breakages.
- **Performance**: Reduced React re-renders by batching UI updates when streaming large DOM trees from the backend.
